## Installation
**Manual:**
```bash
git clone https://github.com/jbxyz/snapchat-username-checker.git
cd snapchat-username-checker
pip3 install -r requirements.txt
python3 snapchat.py
```

**Automatic:**
```
Launch the setup.bat file
```

## Requirements
- Python 3.6+
- requests
- colorama
- pyfiglet
